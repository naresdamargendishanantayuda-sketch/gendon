🎮 Story Cyber Attack

Misi: Selamatkan Akun Sekolah

Kamu menerima pesan WhatsApp dari “Admin Sekolah” yang meminta kamu login lewat link untuk verifikasi nilai. Apa yang kamu lakukan?

Pilih keputusanmu. Jawaban aman akan memberi poin.

🎣 Animasi Phishing: Website Asli vs Palsu

Perhatikan URL dan tanda-tanda bahaya.

https://accounts.google.com/login

✅ Website Asli

  • Domain resmi
  • HTTPS aktif
  • Tidak memakai kata mendesak berlebihan
  • Tampilan konsisten
http://g00gle-login.verify-user-security.xyz

⚠️ Website Palsu

  • Domain aneh
  • Menggunakan HTTP
  • Ada angka mengganti huruf: g00gle
  • Mendesak korban login cepat
Kesimpulan: Jangan hanya percaya tampilan. Selalu cek URL, HTTPS, dan sumber link.
