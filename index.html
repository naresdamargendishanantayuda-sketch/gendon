📚 Materi Cyber Security

1. Phishing

Penipuan yang menyamar sebagai pihak resmi untuk mencuri akun, OTP, password, atau data pribadi.

  • Cek URL
  • Jangan klik link mencurigakan
  • Waspada pesan mendesak

2. Password Security

Password kuat sebaiknya panjang, unik, dan berbeda untuk tiap akun. Gunakan password manager.

3. Malware

Program berbahaya seperti virus, trojan, ransomware, spyware, dan worm.

4. Social Engineering

Teknik manipulasi manusia agar korban memberikan data rahasia secara sukarela.

5. HTTPS

HTTPS mengenkripsi data antara browser dan server sehingga lebih aman dari penyadapan.

6. 2FA / MFA

Lapisan keamanan tambahan selain password, misalnya OTP, authenticator app, atau security key.

7. WiFi Publik

Hindari login akun penting di WiFi publik tanpa perlindungan. Gunakan HTTPS dan VPN jika perlu.

8. Backup

Backup membantu memulihkan data saat terkena ransomware, kerusakan perangkat, atau human error.
