🏆 Leaderboard

Leaderboard lokal tersimpan di browser. Untuk online semua device, sambungkan ke Firebase.

    🔥 Upgrade Leaderboard Online Firebase

    Untuk membuat skor semua orang masuk ke satu leaderboard global, butuh database. Nanti kode Firebase bisa ditempel di bagian script.

    1. Buat project di Firebase.
    2. Aktifkan Firestore Database.
    3. Ambil Firebase config.
    4. Ganti fungsi saveScore() dan renderLeader() agar membaca/menulis ke Firestore.

    🌐 Domain Custom

    Kalau ingin domain seperti gendonlab.xyz, beli domain lalu sambungkan DNS ke GitHub Pages. Untuk sementara link GitHub Pages sudah cukup untuk pameran.

    Gunakan hanya untuk edukasi dan awareness. Jangan menyerang perangkat orang lain.
